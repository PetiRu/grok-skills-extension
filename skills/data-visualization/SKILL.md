# Data Visualization

## Tools & Integration
- Integrate with xlsx skill.
- Use Python (matplotlib, seaborn, plotly) via bash.
- Output images via render components or files.

## Workflow
1. Load data (xlsx, csv).
2. Clean/preprocess.
3. Generate appropriate viz (bar, line, scatter, heatmap).
4. Save as PNG/SVG and render.
